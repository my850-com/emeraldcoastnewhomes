# Emerald Coast New Homes

New construction real estate website for the Emerald Coast of Florida.

## Overview
- **Domain:** emeraldcoastnewhomes.com
- **Platform:** GitHub Pages
- **Purpose:** Generate buyer leads for new construction homes

## Target Areas
- 30A Corridor (South Walton)
- Destin / Fort Walton Beach
- Panama City / Panama City Beach
- Niceville / Crestview
- Navarre / Santa Rosa County

## Featured Communities
- NatureWalk at WaterColor
- The Village at Grayton Beach (55+)
- Prominence Townhomes
- Watersound Origins
- Driftwood Estates (Niceville)
- Coleman Place (Crestview)

## Lead Generation
- Email capture for exclusive updates
- Form submissions routed to Google Sheets
- Direct contact for community-specific inquiries

## Built With
- HTML5 / CSS3
- GitHub Pages hosting
- FormSubmit.co for form handling
